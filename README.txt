## Description

*Quantum Ontologies: Beyond Efficiency in Digital Learning Spaces* is a scholarly, research-based webtext. In this webtext, I argue for seeing digital learning spaces as ontologically multiple and argue for enacting these educational enviroments in more socially just ways. 


This project has been reviewed and approved by an Institutional Review Board (IRB) and has been peer-reviewed for publication by *Kairos*, a leading journal in digital rhetoric.

## Usage

This project is intended to be explored interactively via a web interface. It is designed for scholars, educators, and students interested in digital rhetoric, critical pedagogy, and alternative approaches to instructional design.
